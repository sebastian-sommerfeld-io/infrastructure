. Screen and speakers
. Allow password-less SSH connections

== Quality Goals
. The whole setup is Infrastructure as Code (as much as possible) -> This setup is run partially by hand.

== Install
. From machine "kobol"
.. Write OS to SD card image:ROOT:homelab/physical/tron/tron-sd-card.png[]
. From RasPi node
.. Start RasPi node with SD card
.. Complete setup wizard
.. Adjust appearance (dock size, dark mode, ...)
. From machine "kobol"
.. Run `src/main/provision-nodes.sh` to provision the RasPi node
. From RasPi node
.. Connect to Firefox sync
